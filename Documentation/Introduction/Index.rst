============
Introduction
============

The Tiny PNG extension connects your TYPO3 installation with the
TinyPNG API to compress PNG and JPG images automatically.

Features
========

- Automatic image compression during upload
- Bulk optimization via Scheduler task
- Configurable image resizing
- Exclude specific folders from optimization
- Support for TYPO3 13 LTS
