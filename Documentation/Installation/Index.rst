============
Installation
============

Requirements
============

- PHP: >= 8.2
- Composer: >= 2.4
- TYPO3: 13.x
- Required PHP extensions: gd, curl, json, pdo

Installation Steps
==================

1. Install via Composer:

.. code-block:: bash

   composer require twoh/twoh_tiny_png

2. Activate the extension in the Extension Manager

3. Include the static TypoScript template

4. Get your API key from https://tinypng.com/developers

5. Configure the extension (see Configuration section)
